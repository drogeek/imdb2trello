# imdb2trello
a plugin for web browser to parse imdb pages and update a list on trello

The plugin requires permissions for the trello API, for this, you need:
- your user name 
    This can be extracted from the url when you access your different boards https://trello.com/u/YOUR_USER_NAME/boards
- an API key
    As for now, you need to create a power-up here https://trello.com/power-ups/admin, and then get the API Key
- a token
    you can get it using https://trello.com/1/authorize?expiration=never&name=imdb2trello&scope=read,write&response_type=token&key=YOUR_API_KEY
